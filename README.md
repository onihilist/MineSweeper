# MineSweeper
================

## But du jeu
Le but du jeu est de déminer un champ de mines en cliquant sur les cases pour les ouvrir, tout en évitant de cliquer sur une mine. Le jeu se termine lorsque toutes les cases sans mines ont été ouvertes ou lorsque vous cliquez sur une mine.

## Règles du jeu
* Le jeu se joue sur un champ de cases carré.
* Certaines cases contiennent des mines, tandis que d'autres sont vides.
* Lorsque vous cliquez sur une case, elle s'ouvre et révèle si elle contient une mine ou non.
* Si une case contient une mine, le jeu se termine et vous perdez.
* Si une case est vide, elle affichera le nombre de mines adjacentes.
* Vous pouvez utiliser ces informations pour déduire quels sont les endroits où il est probable que se trouvent les mines.

## Objectif
L'objectif est de déminer le champ de mines en ouvrant toutes les cases sans mines.

## Fonctionnalités
* Génération aléatoire du champ de mines
* Affichage du nombre de mines adjacentes pour chaque case
* Possibilité de marquer les cases suspectes
* Compteur de temps pour mesurer votre rapidité

## Comment jouer
Pour jouer, simplement ouvrez le fichier et commencez à cliquer sur les cases pour les ouvrir. Bonne chance!
